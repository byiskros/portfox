
## Концепция: чистый редактор-документ

Вдохновение: Medium editor, Notion в режиме документа, Bear, iA Writer.

### Визуал

- **Фон страницы**: чистый белый (`background`), без серых подложек и карточек
- **Контент**: 740px по центру, много воздуха сверху и снизу
- **Блоки без контейнеров**: текст, заголовки и картинки идут потоком — как в статье, без рамок и фонов
- **Типографика**: крупный заголовок (36–40px, bold), текст (17–18px, line-height 1.7), heading блоки (24–28px, semibold)

### Верхняя панель

- Минималистичная: «← Back» слева, кнопка «Publish» справа
- Прибита к верху (sticky), полупрозрачный белый фон с blur — как в Notion

### Управление блоками

- **Ховер-меню слева**: при наведении на блок слева появляется иконка `⋮⋮` (drag handle / grip) — но без drag-and-drop, просто как точка входа
- **По клику на `⋮⋮`**: выпадающее меню с действиями:
  - Heading / Text (переключение типа)
  - Move up / Move down
  - Delete
- **Вставка блоков**: между блоками при ховере — тонкая горизонтальная линия с кнопкой «+» по центру. По клику — выбор типа (Heading / Text / Image)

### Поведение

- Enter → новый текстовый блок
- Textarea auto-resize — весь текст виден
- Фокус на новый блок после создания
- Title — без бордеров, просто большой placeholder «Untitled»

### Обложка

- Без рамки, полноширинная картинка внутри контентной области
- При ховере — полупрозрачный оверлей с кнопкой замены

### Итого

Никаких карточек, серых фонов, бордеров на блоках. Чистый документ с невидимыми контролами, которые появляются только когда нужны.
